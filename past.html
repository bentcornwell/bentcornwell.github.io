My Past Experiences with Computing

The webpage was quite basic, it wasn't hosted, lacked a CSS page, yet it served as a great introduction to learning HTML.



This screenshot is from the main page of my first webpage. It features a default white background, with a single image and a few dropdown boxes. This webpage served to be my first experience with HTML and greatly assisted me with learning fundamental skills that I have applied across this website.


Other than my IST classes during highschool, I haven't had much else experience. This assessment task has proved extremely beneficial for me, as it has allowed me to progress my skills in developing a fuctional website utilising both HTML and CSS.
